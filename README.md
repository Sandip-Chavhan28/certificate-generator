# 🎓 Certificate Generator

A web-based **Certificate Generator** built using **HTML, CSS, and JavaScript** that automates the process of creating personalized certificates in bulk. Simply upload a certificate template, import student names from an Excel/CSV file (or paste them manually), customize the text style and position, and generate all certificates in a single ZIP file.

---

## ✨ Features

- 📄 Upload certificate template (PNG/JPG)
- 📊 Import student names from Excel (.xlsx/.xls) or CSV
- ✍️ Paste names manually
- 👀 Live certificate preview
- 🖱️ Click and drag to position the name
- 🔤 Customize font family, size, color, bold, and italic
- 📍 Text alignment (Left/Center)
- 📦 Generate certificates in bulk
- 📥 Download all certificates as a ZIP file
- 📈 Progress indicator during generation
- 📱 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API
- SheetJS (XLSX)
- JSZip

---

## 📂 How to Use

1. Upload a certificate template (PNG or JPG).
2. Upload an Excel/CSV file containing student names, or paste names manually.
3. Click on the certificate preview to position the name.
4. Customize the font, size, color, and style.
5. Click **Generate All & Download ZIP**.
6. Download the generated ZIP containing all certificates.
